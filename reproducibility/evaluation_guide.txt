This guide provides step-by-step instructions for evaluating the trained models

═══ PREREQUISITES ═══

1. Ensure all dependencies are installed:
   ```bash
   pip install -r requirements.txt
   ```

2. Verify trained model checkpoints exist:
   - checkpoints/s_bio_best.pt
   - checkpoints/s_mech_best.pt
   - checkpoints/gflownet_best.pt
   - checkpoints/pipeline_config.json

3. Verify data files exist:
   - data/polymer_smiles_db.py
   - data/real_polymer_data.py
   - data/fransen_polyester_data.py

═══ TROUBLESHOOTING ═══

Issue: "ModuleNotFoundError: No module named 'torch'"
Solution: Install PyTorch: pip install torch==2.0.1

Issue: "FileNotFoundError: checkpoints/gflownet_best.pt"
Solution: Run the full pipeline first: python run_pipeline.py

Issue: "CUDA out of memory"
Solution: This shouldn't happen (CPU-only). Check that GPU is not being used.

Issue: "Validation fails with different scores"
Solution: Ensure random seeds are set correctly (see random_seeds.txt)

Issue: "Figures look different"
Solution: Matplotlib version differences. Update: pip install matplotlib==3.7.1

Issue: "Runtime much longer than expected"
Solution: Check CPU usage. Ensure no background processes consuming resources.
